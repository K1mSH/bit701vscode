git 명령어를 처리하기 위한 git bash를 설치
폴더에서 우클릭으로 열던지 cmd로 접근 가능
----------------------------------------------------------------------------
** 로컬 저장소: 파일이나 디렉토리를 저장하는 장소, 내 컴퓨터에 위치함
                로컬 디렉토리로 저장된 디렉토리안에 .git.이 생성되는데
                그 안에 각종 소스코드나 브랜치등의 정보가 저장됨
                로컬 저장소에 일단 소스코드를 저장하는 명령: commit
** 원격 저장소: 서버나 네트워크에 있는 저장소를 의미
                일반적으로 로컬 저장소에 저장 후 원격 저장소에 저장하는 방식
                우리는 github 라는 원격 저장소로 저장
                명령어는 : push
** 내 컴퓨터의 특정 폴더를 로컬저장소로 지정하는 방법
    git init
** github의 repositorys를 연결 (원격저장소)
    git remote add origin 깃허브주소 // origin은 원격 저장소 이름임
    폴더가 이미 다른 깃주소에 연결되어 있을 경우는
    git remote set-url origin 깃허브주소 
    

** 원격 저장소의 소스코드를 내 컴퓨터에 복사해서 가져오려면
    git clone 원격저장소 경로(깃허브 주소)
** commit
    git commit -m "메세지"
** push
    git push origin branch명
** branch간 이동
    git checkout branch명 // 해당 branch로 이동
** 새 브랜치 생성
    git branch 새브랜치명
** 브랜치 조회
    git branch
** 브랜치  제거
    git branch -D 브랜치명
    git branch 브랜치명 -d
** git log보기
    git log 
** 원격 저장소의 변경된 내용을 로컬 저장소로 가져오려면
    git pull 
    git pull origin branch명
** master branch에 angel branch의 소스코드를 병합하려면
    git checkout master
    git merge angel
